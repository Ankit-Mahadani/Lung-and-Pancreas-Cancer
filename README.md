# 🧬 Lung & Pancreatic Cancer Datasets

This project uses publicly available medical imaging datasets for building deep learning models to detect lung cancer and pancreatic cancer using CT scans and histopathology images.

------------------------------------------------------------

🫁 LUNG CANCER DATASETS

1) IQ-OTH/NCCD Lung Cancer Dataset

Description:
One of the most widely used CT scan datasets for lung cancer classification.

Features:
- ~1100 CT scan images
- 3 classes: Normal, Benign, Malignant
- Suitable for CNN classification and transfer learning

Kaggle Link:
https://www.kaggle.com/datasets/hamdallak/the-iqothnccd-lung-cancer-dataset

------------------------------------------------------------

2) Lung & Colon Cancer Histopathology Dataset

Description:
Large dataset containing histopathology images for cancer detection.

Features:
- 25,000+ images
- Lung benign & malignant classes
- Suitable for deep learning classification

Kaggle Link:
https://www.kaggle.com/datasets/andrewmvd/lung-and-colon-cancer-histopathological-images

------------------------------------------------------------

🧬 PANCREATIC CANCER DATASETS

3) Pancreas CT Dataset

Description:
CT scan dataset used for pancreas detection and segmentation tasks.

Features:
- Medical CT scans
- Useful for segmentation and detection models

Kaggle Link:
https://www.kaggle.com/datasets/salihayesilyurt/pancreas-ct

------------------------------------------------------------

4) Pancreatic Cancer Histopathology Dataset

Description:
Microscopic tissue images for pancreatic cancer classification.

Features:
- Cancer vs Normal classes
- Useful for CNN training and classification

Kaggle Link:
https://www.kaggle.com/datasets/kushagra3204/pancreatic-cancer-histopathology-images

------------------------------------------------------------

Recommended Usage for This Project:

Lung Cancer:
- Use IQ-OTH dataset for CT scan classification
- Use Histopathology dataset for high-accuracy training

Pancreatic Cancer:
- Use Pancreas CT dataset for segmentation
- Use Histopathology dataset for classification
