🫁 IQ-OTH_NCCD Lung Cancer Dataset

This is one of the most popular CT scan datasets used for lung cancer deep learning projects.

📊 Details

Contains ~1100 CT scan images

3 classes:

Normal

Benign

Malignant

Good for:

CNN classification

Transfer learning

Medical image preprocessing
